## 🌟 What is nano-banana-prompts?

Nano-banana-prompts is a tool that helps you create image prompts for AI art. You write your ideas in simple words. The program turns them into expert prompts using Gemini Flash. It then sends these prompts to an image generator API. This generates 4K high-quality images based on your input. The tool also includes Claude Code skills for easier handling and better results.

This tool works well if you want to create AI-generated images but don’t want to write complicated prompts yourself. Everything runs automatically once you provide your idea. It is designed for anyone interested in AI art, from beginners to hobbyists.

## 🛠 How it works

Nano-banana-prompts uses a workflow known as agentic automation:

- You give a raw idea as input.  
- Gemini Flash refines this idea into clear, powerful prompts.  
- The prompts are sent to an AI image-generation service through the API.  
- Claude Code skills help format and manage prompts correctly.  
- The service creates 4K images and sends them back to the app.  
- You view and save the final images.

This flow happens in seconds. You do not need to know about APIs or prompt design. The program handles the complexity.
